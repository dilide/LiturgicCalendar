# 世界各地主显节日期

主显节（Epiphany）通常定在 **1月6日**，但在现代天主教会的礼仪规定中，如果该节日在那一国家不是“法定的公共假日（义务假日）”，主教团通常有权将其挪至 **1月2日至1月8日之间的主日（星期日）** 庆祝。

因此，主显节是“固定”还是“挪动”，主要取决于**该国是否将1月6日定为公共假期**，以及当地教会的规定。

以下是按主要语言/地区划分的详细情况：

## 1. 西班牙语地区（Español）

西班牙语世界是主显节（Día de los Reyes Magos）庆祝最隆重的地区，通常与送礼物紧密相关。

*   **西班牙（Spain）：**
    *   **日期：固定在 1月6日。**
    *   **状态：** 全国公共假日。
    *   **备注：** 这是西班牙极其重要的节日，重要程度甚至超过圣诞节本身（传统上这天是送礼物的日子）。1月5日晚会有盛大的“三王游行”（Cabalgata de Reyes）。
*   **拉丁美洲（Mexico, Argentina, Puerto Rico, etc.）：**
    *   **墨西哥、阿根廷、波多黎各等：** 传统上都在 **1月6日** 庆祝。虽然在很多国家这天不一定是法定公休日（除了波多黎各等地），但文化习俗（如切国王蛋糕 Rosca de Reyes）都固定在1月6日。
    *   **哥伦比亚：** 特殊情况。根据《埃米利亚尼法》（Ley Emiliani），如果节日不在周一，公共假期会挪到 **随后的星期一**。所以虽然宗教意义在1月6日，但放假是在靠近的周一。
    *   **教会礼仪：** 在没有将1月6日列为法定假日的拉美国家，天主教会通常会将**宗教庆典（弥撒）**挪至最近的主日，但民间庆祝活动仍在6日。

## 2. 意大利语地区（Italiano）
*   **意大利（Italy）：**
    *   **日期：固定在 1月6日。**
    *   **状态：** 全国公共假日。
    *   **备注：** 意大利人非常重视这一天，除了宗教意义，还有民俗人物“贝法娜女巫”（La Befana）给孩子送糖果或煤炭。

## 3. 葡萄牙语地区（Português）
*   **葡萄牙（Portugal）：**
    *   **日期：** 宗教庆典通常 **挪至最近的主日**。
    *   **状态：** 目前 **不是** 公共假日（曾被取消）。
    *   **备注：** 虽然主要庆祝活动挪到了周日，但在1月6日当天，人们依然会吃传统的“国王蛋糕”（Bolo Rei）。
*   **巴西（Brazil）：**
    *   **日期：** 宗教庆典 **挪至最近的主日**。
    *   **状态：** 不是全国公共假日。
    *   **备注：** 巴西有“Folia de Reis”（三王节庆典）的民俗传统，通常持续数天，但教会的主显节弥撒是在周日举行。

## 4. 英语地区（English）
英语国家通常不将主显节作为公共假日，因此教会的做法更为灵活。

*   **美国（USA）、加拿大（Canada）、澳大利亚（Australia）：**
    *   **日期：** 天主教会将其 **挪至 1月2日 至 1月8日 之间的主日**。
    *   **状态：** 不是公共假日。
    *   **备注：** 大多数信徒在周日去教堂庆祝。如果是圣公会（Episcopalian/Anglican），有些教区会保留在1月6日举行特别服务，但主力庆祝仍在周日。
*   **英国（UK）：**
    *   **英格兰和威尔士：** 天主教会通常将庆祝活动 **挪至最近的主日**（除非1月6日刚好是周日或某些特定教区的豁免）。
    *   **爱尔兰（Ireland）：**
        *   **日期：固定在 1月6日。**
        *   **状态：** 这一天被称为“小圣诞节”（Little Christmas）或“妇女圣诞节”（Nollaig na mBan）。虽然不一定是全面放假，但爱尔兰天主教会通常将其视为“当守瞻礼”（Holy Day of Obligation），信徒需在1月6日望弥撒，无论它是星期几。

## 5. 其他主要欧洲语言（法语、德语）
*   **法语地区（法国）：**
    *   **日期：** 宗教上 **挪至 1月1日 后的第一个主日**（即1月2日至8日之间）。
    *   **状态：** 不是公共假日。
    *   **备注：** 法国人整个1月份都会吃“国王饼”（Galette des Rois），但这主要是文化习俗，宗教仪式在周日。
*   **德语地区（德国、奥地利、瑞士）：**
    *   **德国：** 视州而定。在巴伐利亚（Bavaria）、巴登-符腾堡（Baden-Württemberg）和萨克森-安哈尔特（Saxony-Anhalt），是 **固定在 1月6日** 的公共假日。在其他州，教会通常挪至周日庆祝。
    *   **奥地利：** **固定在 1月6日**（公共假日）。

## 总结对照表

| 语言/地区 | 主显节日期 (宗教庆典) | 是否为公共假日 | 备注 |
| :--- | :--- | :--- | :--- |
| **西班牙** | **固定 1月6日** | 是 | 三王节游行，最重要的送礼日 |
| **拉美 (大多数)** | 民俗1月6日，礼仪多挪至主日 | 否 (多数) | 墨西哥等地民俗极强，由于不放假，教堂仪式可能在周日 |
| **意大利** | **固定 1月6日** | 是 | La Befana 女巫节 |
| **葡萄牙** | **挪至主日** | 否 | 吃 Bolo Rei 蛋糕 |
| **巴西** | **挪至主日** | 否 | Dia de Reis |
| **美国/加拿大** | **挪至主日** | 否 | 很少有民俗庆祝，主要是教会活动 |
| **英国** | **挪至主日** | 否 | 爱尔兰例外，保留在1月6日 |
| **法国** | **挪至主日** | 否 | 吃国王饼 |

**简单结论：**
*   凡是**天主教传统深厚且将其列为法定假日**的国家（如西班牙、意大利、奥地利），日期都**固定在1月6日**。
*   凡是**不放假**的国家（如美国、巴西、法国、大多数英语国家），教会为了方便信徒参与，通常会将日期**挪至临近的主日**。
